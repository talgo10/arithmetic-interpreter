# talgo/arithmetic-interpreter

## Domain Language
1. Assignments `x = ..`
2. Compound Assignments `x += ..` `x -= ..`
3. Unary Operators `++i` `i++` `-x`
4. Binary Operators `+` `-` `*`
5. Numbers (no floating point) `5` `-2`
6. Variables `x`

## How to Build & Run ?
### Prerequisites
1. Maven 3+
2. Java 8+

### Compile and Run the Application
```
mvn compile exec:java
```

### Running Tests
```
mvn test
```
